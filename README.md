\# Job Portal Project



\## 💼 Description

Full-stack Job Portal web application built using React and Spring Boot.



\## 🚀 Features

\- User Registration \& Login

\- Job Listings

\- Apply for Jobs

\- Recruiter Job Posting

\- Modern UI



\## 🛠 Tech Stack

\- React.js

\- Spring Boot

\- MySQL



\## 📁 Structure

\- Backend → Spring Boot APIs

\- Frontend → React UI



\## ▶️ Run Project



\### Backend

cd Backend/SearchJob  

mvn spring-boot:run  



\### Frontend

cd Frontend/frontend  

npm install  

npm start
